The Problem

Lots of mobile operating systems

  • Android, iOS,
  • Windows Phone 7 + 8
  • Blackberry
  • Firefox OS, Tizen, Ubuntu, Amazon FireOS

The Problem

  • Learn multiple languages or hire multiple teams
  • Need to develop on Mac & Windows
  • Huge amount of investment to target multiple platforms

Solution?

Apache Cordova!

Apache Cordova allows developers to build cross platform mobiles apps using HTML, JS & CSS

History of Cordova

2008 - PhoneGap was created

iPhone sdk, Blackberry 4.2 and Android 1.0

2009 - Accept & Reject

As developers started to love us, Apple did not

2010 - Maturing

Outside contributions picked up (including IBM!)

2011 - Growing Pains

1.0, Adobe, Apache

2012 - Cadence realized

2.0, and 1 million downloads.

2013 - Workflow overhaul

3.0, broke out plugins, cordova-cli distribution moved to npm
